<a id="introduction"></a>
## Introduction

libstrings provides functions that can manage a list of unique strings.  The strings are indexed in an AVL tree by string value and also by unique id value.

<a id="installation"></a>
## Installation

In the spirit of GNU Autotools, follow these steps to install libstrings:

  1) Obtain libstrings.txz
  2) tar xf libstrings.txz
  3) cd libstrings
  4) make
  5) sudo make install

## Library

A static library named libstrings.a should be installed in /usr/local/lib after running `sudo make install`
